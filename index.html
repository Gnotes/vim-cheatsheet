Vim 常用命令速查卡片

📝 模式切换

i 插入模式 (光标前)
I 插入模式 (行首)
a 插入模式 (光标后)
A 插入模式 (行尾)
Esc 返回普通模式

📖 文件操作

:w 保存文件
:q 退出
:wq 保存并退出
:q! 强制退出不保存
:e filename 打开文件

🔍 光标移动

h 左移
l 右移
0 行首
^ 行首第一个非空字符
$ 行尾
w 下一个单词开头
e 单词结尾
b 单词开头
gg 文件开头
G 文件结尾
:n 跳到第 n 行

✂️ 删除与剪切

x 删除光标字符
dd 删除整行
dw 删除一个单词
d$ 删除到行尾
d0 删除到行首
D 删除到行尾

📋 复制与粘贴

yy 复制整行
yw 复制一个单词
y$ 复制到行尾
p 粘贴到光标后
P 粘贴到光标前

✏️ 替换与修改

r 替换光标字符
R 替换模式
cw 修改一个单词
cc 修改整行
C 修改到行尾

🔄 撤销与恢复

u 撤销
Ctrl + r 恢复撤销

🔎 搜索与替换

/word 向下搜索
?word 向上搜索
n 下一个匹配
N 上一个匹配
:%s/old/new/g 全文替换
:s/old/new/g 当前行替换

📑 多窗口与分屏

:split filename 水平分屏
:vsplit filename 垂直分屏
Ctrl + w + w 切换窗口
Ctrl + w + q 关闭窗口
Ctrl + w + h/j/k/l 窗口间移动
